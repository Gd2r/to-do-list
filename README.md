# Hisham's Todo List

A modern task manager that helps you plan your day, manage your projects, and make real progress toward your goals.

## Features
- Create and manage multiple projects
- Add tasks with priorities and due dates
- Mark tasks as complete
- Delete tasks and projects
- Responsive design
- Local storage persistence
